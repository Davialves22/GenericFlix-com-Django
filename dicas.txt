pip install django > instala o django

No terminal da pasta do projeto, digite:

python -m django startproject genericFlix .


O ponto (.) no final cria o projeto na pasta atual, sem criar subpastas extras.

Isso funciona mesmo que django-admin não seja reconhecido pelo Windows.

Depois, para rodar o servidor:

python manage.py runserver

django-admin startapp > cria um app

python -m django startapp filme > cria um app sem precisar de admin

migrate e o que torna o banco de dados atualizado > python manage.py migrate

python manage.py makemigrations > cria a estrutura de migração

sempre que fizer alguma mudança tem que rodar os dois codigos

criar um super user > python manage.py createsuperuser

Para formatar automaticamente um arquivo ou bloco de código:

No Windows/Linux: Ctrl+Alt+L
